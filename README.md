# Sohrab Fadai

Founder of [Heartspace](https://heartspace.ai) – a Stockholm-based PR and public affairs agency.

I use code to solve communications problems: press distribution systems, client pressrooms, survey platforms, and tools that make PR less manual and more precise.

## What I'm building

- **Pressroom platform** – Astro + Sanity + Cloudflare Pages. A faster, leaner alternative to legacy PR platforms, currently in production with clients.
- **PR-collaboration tool** – Remix + Supabase + TipTap. An AI-powered editor and collaboration workspace tied into everything else. 
- **Email distribution engine** – Resend-based system with dynamic merge tags, conditional logic, AI-magic, and subdomain delegation for client sending.
- **Survey portal** – Tiered product for polling data, built on a Novus partnership.
- **[sohrab.se](https://www.sohrab.se)** – Personal site, Astro.

## Stack

Astro · Sanity · Cloudflare Pages · Resend · Vercel · TypeScript · React · Tailwind · Supabase · Remix.js  

## Background

17 years in Swedish public affairs. Rhetoric degree from Uppsala University. Author of *Etisk försäljning*. I think a lot about how language shapes opinion – and how technology can make that process more honest.
